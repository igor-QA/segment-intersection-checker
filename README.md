# segment-intersection-checker
Инструмент для проверки пересечения двух отрезков. Включает реализацию и тесты.

Стек: Java, JUnit5, maven

Сборка

```
mvn clean package
```

Запуск

```
java -jar target/segment-intersection.jar
```

Вывод

```
Segments intersect: true
```

Запуск тестов

```
mvn clean test
```

Allure отчет 

```
allure: serve и allure: report
```
